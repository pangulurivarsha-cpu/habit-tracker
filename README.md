# Sports Habit Tracker

A mobile-first habit tracker built with React and Vite, featuring a premium red gradient glassmorphism UI.

## Features
- **Login**: Mock authentication with Mobile/Email.
- **Dashboard**: Select sports to track (Tennis, Badminton, Pickleball, Basketball).
- **Tracker**: Monthly calendar view with daily check-ins.
- **Stats**: View Weekly and Monthly total counts.
- **Responsive**: Design optimized for mobile devices.

## Setup Instructions

### Prerequisites
- Node.js (v16 or higher)
- npm (Node Package Manager)

### Installation
1.  Open your terminal/command prompt.
2.  Navigate to the project directory:
    ```bash
    cd habit-tracker
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App
1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open the link shown in the terminal (usually `http://localhost:5173`).

## Troubleshooting
-   If `npm install` fails, ensure Node.js is correctly installed and added to your system PATH.
-   The app uses `lucide-react` for icons and `react-router-dom` for navigation.
